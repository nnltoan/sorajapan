2. Phạm vi Công việc (Scope of Work)

Giao diện (Frontend & UI/UX)

Hạng mục Mô tả
Thiết kế Chuẩn mực Tone màu Xanh/Cam đồng bộ nhận diện, CSS grid chuẩn, Font hiện đại.
Responsive 100% Tương thích hoàn hảo mọi thiết bị di động Mobile-first.
Interactive UI Slider Hero Section, Animation Floating Buttons (Zalo, Msg, Call).
Hệ thống 20+ Pages Điều hướng thông minh, Cards lưới phân bố dịch vụ logic, thân thiện với SEO.

Công nghệ Xử lý (Logic & Data)

Hạng mục Mô tả
Google Sheets Board Viết logic Parser CSV real-time để render thẻ Đơn Hàng (Không cần server DB).
Bộ lọc JS Động Lọc đơn hàng theo chứng chỉ JLPT N5-N1 tức thì ngay trên trình duyệt.
Webhook Contact Form Bắn Data điền Form của khách hàng đẩy thẳng vào Google Sheet của bộ phận CSKH.
Tối ưu Load Speed Kiến trúc tinh gọn, cache-busting, bypass CORS tự động.
